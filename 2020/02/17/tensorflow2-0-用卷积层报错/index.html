tensorflow2.0 用卷积层报错

发表于 2020-02-17 | 分类于 tensorflow | 浏览 次

报错信息

尝试解决办法：

安装可行版本的CUDA和cuDNN，发现问题仍未解决。

怀疑有可能是显存不足，打开nvidia msi 查看，发现tensorflow一下子把显存占完了，导致后面用卷积层时无法初始化cuDNN

解决办法

我们只要把tf设置为按需申请显存就行了．
